# ice-yunqi-homepage

浙江强驰电力

> 使用文档

使用:

- 启动调试服务: `npm start`
- 构建: `npm run build`

目录结构:

- react-router @4.x 默认采用 hashHistory 的单页应用
- 入口文件: `src/index.js`
- 导航配置: `src/menuConfig.js`
- 路由配置: `src/routerConfig.js`
- 路由入口: `src/router.jsx`
- 布局文件: `src/layouts`
- 通用组件: `src/components`
- 页面文件: `src/pages`
